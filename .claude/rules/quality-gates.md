# Quality Gates

## Gate 1 — Intelligence Completeness
**Must pass before Phase 5 (Context Engineering)**
- [ ] Target decision-maker profile is substantive with cited sources
- [ ] Source entity's thesis is clear and compelling in business language
- [ ] Financial narrative has specific numbers with stated assumptions
- [ ] Technical data is interpreted in non-technical language
- [ ] At least 10 objections identified with sourced responses
- [ ] Regulatory pathway is documented with current status
- [ ] Cultural protocol is documented with specific guidance
**Process:** Lead Agent reviews all processed/ files. Gaps trigger targeted research.

## Gate 2 — Source Document Quality
**Must pass before Phase 6 (Production)**
- [ ] Each document readable by non-technical executive in 5 minutes
- [ ] Each document tells a story, not just presents facts
- [ ] All 9 documents are internally consistent
- [ ] No unverified claims (everything tagged with confidence level)
- [ ] Documents 1-5 + 8-9 are 3-5 pages each
- [ ] Document 6 captures decision-making style and priorities
**Process:** Opus reviews each doc. Human reviews docs 1, 2, 3, 5.

## Gate 3 — Deliverable Quality
**Must pass before declaring "final"**
- [ ] All three verification layers passed (source trace, cross-model, adversarial)
- [ ] Slides readable on projector at 10+ feet
- [ ] Tone: confident not arrogant, grounded not timid
- [ ] Zero spelling/grammar errors
- [ ] Cultural sensitivity verified against protocol notes
- [ ] Website loads clean, no broken links
- [ ] Leave-behind prints clean
**Process:** Verification system runs all layers. Human does final review.

## Three-Layer Verification
**Layer 1 — Source Tracing (every claim):**
Identify source doc → trace to original raw intelligence → confirm accuracy → tag verified

**Layer 2 — Cross-Model (critical claims):**
Extract claim → submit to DIFFERENT model → compare → investigate discrepancies

**Layer 3 — Adversarial (final deliverables):**
Fresh Claude session → provide deliverable + objection map → "Find every weakness" → address all issues
